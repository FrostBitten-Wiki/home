Editing: Learn the Basics
Welcome, future contributor! This is the Part 1 of the Editing Guide. This part introduces some basic understanding on how to edit a Wiki Page with HTML, including how to properly create sections, and other stuff.
Before we start...
An Github Account is required in order to contribute changes/save wiki changes via Pull Requesting.
Types of Editing #types-of-editing
There are two types of editing you can use to edit the wiki. It can be Local Editing or Online Editing

Local Editing is the best option you can use to edit the wiki quickly, with that requiring you to install Visual Studio Code (/download, ~90MB), Python (/downloads, ~25MB) (and 2 pip libraries) and git (/downloads ~58MB).

Online Editing is not the best, although good if you dont want to install ~173MB of programs. The online editor can be accessed here: vscode.dev.

For this entire guide, We will be Locally Editing the wiki we want to create edits from!
Preparing the Programs #preparation
Okay, assuming that you have all of the programs installed on your Downloads folder. Follow the stuff below:

Install Visual Studio Code normally.
Install Python, make sure to add it to PATH. There should be a button somewhere (or in the end of installation).
Install git. on the 4th page, Override the default branch name for new repositories as "main".

Aaaaanddd... that's all!
Setting up Visual Studio Code #setup-vsc
Before we setup Visual Studio Code, you must fork a Wiki Repository first. We will be forking the Dummy Wiki (click to fork). The dummy wiki contains almost absolutely nothing... except some templates you can edit from so you can start doing whatever you want or just follow this entire guide.

Now that you've cloned the Dummy Wiki, you can now open Visual Studio Code, and press CTRL + SHIFT + ` to open up a new terminal and type in the following commands:

git config --global user.name "YourGithubName" git config --global user.email "youremail@example.com" git clone https://github.com/[your_githubname]/dummy.git . && git pull

Before we start editing, we must install 2 pip libraries before we run main.py type the following command on the terminal:

pip install fastapi uvicorn

Now that you have those two pip libraries installed, you can now run main.py by typing the following command on the terminal:

python main.py

For Linux computers:

python3 main.py

Alright... now that everything is almost setup, we can actually start editing!!
Now, the Editing part!! #editing-1
To start editing, let's open up a file we want. For example, let's open up home.html. and edit our first Text Element.

<div style="margin-bottom: 20px;"> <text style="font-size: 35px;">XD</text> </div>


Try editing that XD text between > and < to something else from the home.html file, then visit this localhost page http://localhost/dummy to see if the text changed to something else!

You can also edit the font size style to something else, like 50px, or 69%!

<div style="margin-bottom: 20px;"> <text style="font-size: 50px;">XD</text> </div>

<div style="margin-bottom: 20px;"> <text style="font-size: 69%;">XD</text> </div>


You can also change it's color to something else, the color value can be Hex Values ( #696969 ), RGB ( rgb(69, 69, 69) ) and more!

<div style="margin-bottom: 20px;"> <text style="color: #696969; font-size: 69%;">XD</text> </div>

<div style="margin-bottom: 20px;"> <text style="color: rgb(69, 69, 69); font-size: 69%;">XD</text> </div>


Oh yeah, when editing the style value, it's very important that you don't forget " : " (colon) after every property name and " ; " (semicolon) after a property value. Forgetting to put that may cause the Text Element's style to not display properly.
